☀️ SunSys ERP Portal
A robust Human Resource (HR) and Task Management System designed to streamline workflow, task tracking, and department-level communication.

🚀 Overview
The SunSys ERP is a centralized platform developed to bridge the gap between HR administration and employee execution. It features a dual-role interface that allows for granular control over task lifecycles and secure data management.

✨ Key Features
🛠️ HR Command Center (Admin)
Dynamic Task Assignment: Assign tasks with specific descriptions, departments, and priority levels.

Deadline Management: Set precise due dates and time limits for every task.

Multi-Format Resource Attachment: Upload PDF briefs, Excel sheets, videos, or images as task resources.

Real-time Analytics: Dashboard with metrics for Total, Completed, Pending, and Overdue tasks.

Employee Directory: Full CRUD (Create, Read, Update, Delete) operations for team member management.

💻 Employee Department Center
Personalized Task View: See only tasks assigned to your specific department and username.

Progress Tracking: Update task status (Pending, In Progress, Work Completed) and add progress notes.

Work Evidence: Upload proof of work in multiple formats, including images, videos, and documents.

Security: Dedicated security tab allowing employees to update their own passwords independently.

🛠️ Tech Stack
Frontend/UI: Streamlit (Python-based Web Framework)

Database: SQLite3

Data Processing: Pandas

Styling: Custom CSS & HTML Components
